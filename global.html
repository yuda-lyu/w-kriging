Methods

(async) WKriging(arrSrc, arrPred, optopt) → {Promise}

Description:

  • Kriging內外插值，支援2D與3D

Source:
  • WKriging.mjs, line 118
Example
async function test2d() {

    let psSrc = [
        [-0.1, -0.1, 0],
        [1, 0, 0],
        [1, 1, 10],
        [0, 1, 0]
    ]
    let psTar = [
        [0.1, 0.95],
    ]
    let opt = {
        variogram_model: 'gaussian',
        nlags: 9,
    }

    let r = await WKriging(psSrc, psTar, opt)
    console.log('test2d', r)
    // => test2d [ 1.8997805977145759 ]

}
test2d()
    .catch((err) => {
        console.log('catch', err)
    })

async function test3d() {

    let psSrc = [
        [-0.1, -0.1, -0.1, 0],
        [1, 0, 0, 0],
        [1, 1, 0, 0],
        [0, 0, 1, 0],
        [1, 0, 1, 0],
        [1, 1, 1, 10]
    ]
    let psTar = [
        [0.1, 0.1, 0.95],
    ]
    let opt = {
        variogram_model: 'gaussian',
        nlags: 9,
    }

    let r = await WKriging(psSrc, psTar, opt)
    console.log('test3d', r)
    // => test3d [ 1.666666666666666 ]

}
test3d()
    .catch((err) => {
        console.log('catch', err)
    })
Parameters:
Name Type Attributes Default Description
arrSrc Array

輸入觀測數據陣列，可輸入二維座標加觀測數據陣列[[x1,y1,v1],[x2,y2,v2],...]，或三維座標加觀測數據陣列[[x1,y1,z1,v1],[x2,y2,z2,v2],...]
arrPred Array

輸入預測數據陣列，可輸入二維座標陣列[[px1,py1],[px2,py2],...]，或三維座標加觀測數據陣列[[px1,py1,pz1],[px2,py2,pz2],...]
opt Object <optional>
 {}

輸入設定物件，預設{}

Properties

輸入變異圖模式字串，可選'linear'、'power'、'gaussian'、'spherical'、'exponential'、'hole-effect'，其定義詳見pykrige，預設'gaussian'
nlags Integer <optional>
 9

輸入變異圖統計直條圖數量整數，其定義詳見pykrige，預設9
